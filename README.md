# C&M Tires and Muffler

Landing page for **C&M Tires and Muffler**, an auto care center located in Wylie, TX. Built with Next.js 14, TypeScript, Tailwind CSS v4, and Framer Motion.

## Sections

- **Hero** — Full-screen parallax background with rotating images, business name, and CTA
- **Services** — 7 service cards with icons and descriptions in a responsive grid
- **Gallery** — Bento-box style image grid showcasing work
- **Schedule** — Weekly hours table with current day highlighted
- **Contact** — Google Maps embed, WhatsApp, Facebook, phone, address, LGBTQ+ friendly badge
- **Footer** — Nav links, contact info, back-to-top

## Features

- **Bilingual** (English/Spanish) — auto-detects from browser, saves preference to localStorage, toggle via flag buttons in navbar
- **Animated** — Framer Motion scroll reveals, hover effects, parallax hero
- **Responsive** — Mobile-first, hamburger nav on small screens
- **Color palette** — `#B61516` red, near-black, and `#D0D0D0` gray

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
